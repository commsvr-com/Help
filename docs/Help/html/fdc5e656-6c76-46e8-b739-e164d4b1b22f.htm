Starting

After starting the application, on the screen appears splash screen:

Figure: Logo

Then appears the main window:

Figure: Main window

Controls in the main window:

  1. TreeView panel, where all Sandcastle Help File Builder project elements are shown

  2. Information panel, where information about help element (image, code entity, external link, token, topic) are shown.

  3. Checkbox (FormattedLink), if the box is checked, the link will be in formatted form.

  4. Menu bar with “File” menu and “Help” menu.

  5. External link panel, where external links can be created.
