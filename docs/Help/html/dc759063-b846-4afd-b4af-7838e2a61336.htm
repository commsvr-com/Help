Creating of the external link

To create an external link, the “Create external link” panel in main window (5) should be opened. It looks like below:

Figure: Create external link

Controls in “Creating external link” panel:

  1. “Create” button after clicking copies the link in system clipboard and saves it in the file on hard drive.

  2. “Clear” button removes data from the fields: “Link text”, “Link alternate text”, “URI”.

Clear fields

After clicking the “Clear” button (2) all fields become empty:

Figure: Clear
After clicking the "Create" button

After clicking “Create” button, if the link was created properly for the first time, message box appears:

Figure: Link created properly and copied to the file

If the link is created second time, it will be created, but it will not be copied to the file:

Figure: Link created properly and not copied to the file

There might be also the situation, when some of the fields are empty:

Figure: Empty field
